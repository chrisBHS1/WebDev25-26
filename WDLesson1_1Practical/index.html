Web Technologies

HTML

HTML (HyperText Markup Language) is the standard markup language used to create and structure web pages and their content. It uses a system of tags to define elements like headings, paragraphs, and images, which web browsers interpret to display content in a structured layout. HTML provides the basic structure for every website, and is typically used in conjunction with CSS for styling and JavaScript for adding interactivity.

CCS

CSS, which stands for Cascading Style Sheets, is a stylesheet language used to describe the presentation and styling of a document written in a markup language such as HTML or XML. It's a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.

JavaScript

JavaScript (JS) is a high-level, dynamic, and interpreted programming language primarily known for its role in creating interactive and dynamic content on websites. It is one of the three core technologies of the World Wide Web, alongside HTML (for structuring content) and CSS (for styling content).
Bonus

Bootstrap

Bootstrap is not a programming language but a free and open-source front-end framework that uses HTML, CSS, and JavaScript to build websites. Its purpose is to help developers create modern, responsive, and mobile-first websites faster and more efficiently by providing a large collection of reusable components. These components include pre-designed templates for navigation bars, buttons, forms, and a powerful grid system, which allows websites to automatically adapt to different screen sizes.
